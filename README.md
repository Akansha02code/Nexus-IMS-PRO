## ðŸš€ Getting Started

### Prerequisites
- Node.js (v16+)
- JDK 17+
- MySQL Server

### Backend Setup
1. Navigate to `ims-backend`.
2. Update `application.properties` with your MySQL credentials.
3. Run: `./mvnw spring-boot:run`

### Frontend Setup
1. Navigate to `ims-frontend`.
2. Run: `npm install`
3. Run: `npm run dev`
